Insights into Public Perception: Black Lives Matter Campaign Analysis

Project Overview

One of the topics I learnied during my master's degree program was the use of the Spark platform for efficient data processing. One of the assignments given at that time was to implement Spark in real-world situations.

At that time, the Black Live Matter campaign was happening worldwide. It caught my attention, so I chose to address this topic by utilizing Spark to gather data from Twitter social media as the main data sourc.

Challenges

Not only did I use Spark, but I also chose to utilize SparkNLP, a library specifically designed for processing data related to NLP. Initially, using both felt challenging because I didn't have prior experience. It took some time to learn both before I could efficiently implement them into code. And I needed to complete this step quickly before losing the momentum related to the Black Live Matters campaign on social media.

Process

1. Exploration

Read and learn Spark documentation.

2. Development

Write the code in Python. Basically the code will process below:

1. Collect the data from Twitter

2. For every new data entry, the system will run a sentiment analysis using Stanford NLP model

3. Store the Twitter and sentiment result into database

3. Analysis

Analyze the collected data and the sentiment results.

Result

The final outcoume of this project was the of the research in the international conference titled IOP Conference Series: Materials Science and Engineering. Additionally, I had the opportunity to present as a speaker at the conferece, sharing my experience in using Spark and SparkNLP to analyze data from the Black Live Matter campaign. Find the publication here.
