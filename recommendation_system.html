A Robust Recommendation System for Undergraduate Program Selection

Project Overview

This project represents the culmination of my master's degree studies. I developed a recommendation system employing a machine learning approach aimed at assisting high school students in selecting the most suitable undergraduate program based on their academic skills. The core of the model relies on a multi-stages classification model utilizing the Random Forest algorithm. Additionally, I created a user-friendly dashboard using the Dash library to provide users with an intuitive interface for interacting with the recommendation system.

Challenges

One of the project's most challenging aspects revolved around managing the data. In my case, the available data exhibited imbalance, significantly impacting the performance of the data model. To address the issue, I choose to implement SMOTE (Synthetic Minority Oversampling Method) to rebalance the dataset and enhance model accuracy.

Process

1. Data Preparation

This stage encompasses the data collection process, standardization, addressing imbalance issues, and data selection.

2. Development & Model Comparison

The model was developed using Python, with careful consideration given to various aspects throughout the development process. A comparison was made between single-stage and multi-stages models to assess performance differences. Additionally, three commong classification algorithms - Logictic Regression, SVM (Support Vector Machine) and Random Forest - were tested to determine the most suitable approach.

3. Presentation

In addition to the academic paper, I developed a dashboard to showcase the functionality of the recommendationn system. This dashboard was created using the Python library Dash.
