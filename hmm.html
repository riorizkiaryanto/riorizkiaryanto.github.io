Unleashing the Potential of Hidden Markov Model in Named Entity Recognition for Indonesian Text

Project Overview

The project revolves around Named Entity Recognition (NER) using the Hidden Markov Model (HMM) algoritm. As a Jr. Data Scientist within an R&D team, our objective is to conduct research on the Hidden Markov Model algorithm. This algorithm will serve as the core of the product we are developing. The product aims to gather data from various social platforms and provide information on entities currently tranding in discussions. These entities may include names of individuals, locations, events and time frames. In my role as Jr. Data Scientist, I collaborate with Data Engineers to build the data model. On daily basis, I engage in research, purpose testing, fine-tuning the model, and documenting progress.

Challenges

The primary challenge lies in developing an effective data model utilizing the Hidden Markov Model algorithm for Named Entity Recognition. Additiionally, ensuring the model accurately identifies and extracts relevants entities from social media discussions poses a significant challenge.

Process

The process involves collaborative efforts between Data Scientist and Data Engineer to build and refine the data model. Daily tasks include research, purpose testing, model fine-tuning, and documentation of findings. The iterative process allows for continuous improvement and optimization of the model's performance.

Result

Although the data model is still in the development stage, initial results are promising. The model demonstrated the ability to identify several expected entities, showcsasing progress towards achieving the project objectives. Further refinement and testing are ongoing to enhance the model's accuracy and effectiveness.

Data Stack

The project utilized a data stack consisting of Python, MySQL, and RegEx
